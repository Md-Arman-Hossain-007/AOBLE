AMLtab

AML Screening Tool for PEP, Sanctions & Watchlists

Features:

  • 3.3M+ names from OpenSanctions database
  • Fuzzy matching with confidence scores
  • Single & batch screening via API
  • PDF/CSV report downloads
  • Web interface for manual checks
